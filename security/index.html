OWASP Top 10 (2021)

Checklist snapshot

  • Broken Access Control: enforce least privilege and server-side checks.
  • Cryptographic Failures: TLS everywhere, proper key rotation.
  • Injection: parameterized queries, strict input validation.
  • Insecure Design: threat modeling early; safe defaults.
  • Security Misconfiguration: hardened headers, minimal services.
  • Vulnerable Components: patch cadence and SBOM awareness.
  • Identification & Auth Failures: MFA-ready flows, secure session handling.
  • Software & Data Integrity: signed dependencies; CI integrity checks.
  • Logging & Monitoring: structured logs, alerting thresholds.
  • SSRF: deny-list internal metadata, vetted outbound calls.
